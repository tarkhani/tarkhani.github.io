    • Projects

    • Improving caching efficiency of transient data in the vehicular named data network (Master’s thesis): My thesis focuses on designing a caching policy for vehicle storage that is better suited to handle transient data. Transient data is short-lived and only valid for a specific duration. These data also have strong regionality, meaning they are only useful for vehicles passing a specific location. For example, a high-definition map of a road is only useful for vehicles near that road. My caching policy efficiently uses cache size to reduce transmission costs by satisfying requests with vehicle-to-vehicle communication.
    • SD-WAN simulation in EVE-NG: For the final project of the Advanced Computer Networks course, I simulated a Cisco SD-WAN network in EVE-NG, a network simulation tool. The goal of the project was to design and implement a software-defined network and perform traffic engineering.
    • Designed and simulated a small to medium network using OMNeT++: As the final project of the Computer Network Design course, I designed and evaluated a small to medium network using the INET library in OMNeT++. The project focused on aspects like security, routing, traffic engineering, addressing, availability, QoS, and management, following Cisco's PPDIOO methodology.
    • Developed a price prediction model trained on data gathered through web scraping: Using the Selenium library, I extracted data from Divar, a popular resale website. I then preprocessed this data to create a deep learning model for price prediction.
    • Developed a vehicle detection model trained on the vehicles-OpenImages dataset: Utilizing TensorFlow, I developed a vehicle detection model to accurately count the number of cars in a video, serving as an indicator of traffic density.
    • Game Engine: Implemented components of a game engine using OpenGL to analyze techniques for creating believable 3D environments in video games. This included normal mapping, diffuse lighting, specular lighting, and using DuDv mapping to simulate water ripples.
    • Job Portal Application: Developed an Android job portal application that allows job seekers to find suitable positions by posting resumes and browsing job advertisements.
    • PDF to Audiobook using Azure services: Developed a program that converts PDFs into audiobooks, exploring Azure cloud services such as Cognitive Services, Text Analytics, and Blob Storage.
    • Supervised Audio Classification: Implemented a supervised audio classification model. This involved pre-processing audio datasets, extracting MFCC features, and training a convolutional neural network to classify music genres.
    • Web Information Retrieval System: Implemented an information retrieval system based on keyword search and positional indexing for retrieving web-based information.
